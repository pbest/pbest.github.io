The Grid

Basics

The Grid relies on two classes, you can think of the top-level .Grid class as the equivalent to Foundation's row class. It’s the flexbox container that will hold all the columns. If we just adds a max-width it’ll behave as your standard grid. The .Grid-cell class on the other hand is Foundation’s equivalent of the column class. By default a Grid-cell will take on 100% of it’s parent’s width. With extra classes you can specify custom and responsive widths. For more examples and information you can find the docs for the Grid component here.

Modify the grid

To modify the grid’s maximum width and gutter size. Take a look at the --Grid-gutter-size and --Grid-max-width variables on the :root object in app.css.
